First article and Entry for r/dataisbeautiful DataViz

January 20, 2018

As first article on this R and dataviz dedicated blog, I present to you my entry for the DataViz battle of January 2018.

This month and the data source is a .htm webpage containing several algae growthrates according to light and temperature conditions.

For this project I used : 1. R for * Data importatation * Data wrangling and cleaning * Final analysis

  1. Tableau Public
  • Data exploration
  • Detailed data dashboard

Download database and import into R

algae <- htmltab("http://aquatext.com/tables/algaegrwth.htm",2,header = 0)
## No encoding supplied: defaulting to UTF-8.
## Warning: No header generated. Try passing information to header or colNames
algae <- algae[-3,]

algae <- t(algae)

colnames(algae) <- algae[1, ]
algae <- algae[-1,]

algae <- algae %>% as.data.frame() %>% gather(key="specy", value="Growthrate", `Caloneis schroderi`:`Thalassiosira sp.`)
## Warning: attributes are not identical across measure variables;
## they will be dropped
colnames(algae) <- c("Temperature","Light","Specy","GrowthRate")

algae$GrowthRate <- as.numeric(algae$GrowthRate)
## Warning: NAs introduced by coercion
filter(algae,is.na(GrowthRate))
##   Temperature Light                   Specy GrowthRate
## 1          10  5000 Isochrysis aff. galbana         NA

Unexpected value for Isochrysis aff. galbana at Temperature 10, Light 5000 : 0..06. This value is put appart for the rest of the analysis.

Additional information

algae <- algae %>% mutate(Genus=word(algae$Specy,1))

algae <- algae %>% select(Specy,Genus,Temperature,Light,GrowthRate)
algae_means <- algae %>% group_by(Specy,Light) %>% mutate(growth_minus_Temp =mean(GrowthRate, na.rm=T)) %>% ungroup() %>%  group_by(Specy,Temperature) %>% mutate(growth_minus_Light =mean(GrowthRate, na.rm=T)) %>% ungroup() %>% group_by(Specy) %>% mutate(total_mean=mean(GrowthRate, na.rm=T))

write.table(algae_means, "D:/Documents/#02 PROJECTS/MINING/Dataviz Battle/algae.csv", sep=";",dec=".", row.names = F)
