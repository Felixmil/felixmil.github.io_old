How I built this blog

December 23, 2017

Material

Used : https://tclavelle.github.io/blog/blogdown_github/ https://bookdown.org/yihui/blogdown/github-pages.html

How to

1 create 2 repo username/source username/username.github.io 2 clone repo on computer and remove readme files

2 start new R project 3 project option -> website, select source as base directory 4 load necessary libraries

# Load blogdown package and install Hugo
library(blogdown)
install_hugo()

5 choose a theme 6 set “source” as working directory 7 create site

# Create new site in our recently cloned blogdown repository
new_site(theme = "kakawait/hugo-tranquilpeak-theme",
         format = 'toml')

8 add folowing lines in config.toml and update the url

publishDir = "X:/.../username.github.io"
themesdir = "X:/.../source/themes"
baseURL = "https://username.github.io/"

9 build the site

build_site()

10 in Terminal, type

cd X:/.../username.github.io

git add *

git commit -m "update ID"

git push origin master

11 go to https://username.github.io/ in browser
